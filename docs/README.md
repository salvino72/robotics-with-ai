# 📚 Documentazione Ufficiale

Oltre alle lezioni pratiche fornite in questo corso, la comprensione tecnica avanzata si basa sulla consultazione delle guide ufficiali. 
Ecco i riferimenti da consultare ogni qualvolta abbiate dubbi sul funzionamento di un componente, sulla sintassi di una funzione o sull'elettronica di una scheda:

1. **[Documentazione Ufficiale Arduino (docs.arduino.cc)](https://docs.arduino.cc/)**
   Il vero testo di riferimento:
   - Specifiche tecniche per ogni scheda (es. voltaggi tollerati da Arduino UNO, Nano, Mega).
   - "Pinout" interattivi per scoprire la funzione di ogni piedino hardware.
   - Linguaggio di programmazione: il dizionario completo (Reference) di tutte le funzioni C/C++ utilizzabili.
   - Esempi di test ("Getting Started") forniti dall'azienda per verificare il funzionamento dei moduli di base.

2. **File Hardware in Open Source**
   Arduino rilascia tutti i progetti hardware come Open Source. Selezionando dal portale ufficiale la scheda che state utilizzando, potrete scaricare liberamente:
   - I file dello sbroglio elettronico (Gerber).
   - Gli schemi logici dei circuiti stampati.
   - I modelli 3D completi, qualora abbiate bisogno di modellarci intorno dei supporti plastici o pezzi di un telaio robotico tramite stampa 3D (es. usando CAD come Fusion 360 o Tinkercad).

---

> 💡 *Suggerimento: Un buon ingegnere robotico non memorizza mai ogni singola funzione o componente "a memoria", ma sa esatttamente **dove e come** trovare e leggere tempestivamente la documentazione tecnica (Datasheet).*
